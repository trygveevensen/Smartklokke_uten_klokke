# Smartklokke_uten_klokke

Dette prosjektet går ut på å lage en et hjelpemiddel for blinde og synshemmede som monteres på armen til brukeren. Enheten varsler brukeren i form av vibrasjon ved hindringer.
Enheten har også et kamera som kan brukes til å analysere bilde med KI og fortelle brukeren hva som er foran dem.
